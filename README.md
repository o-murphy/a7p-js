# a7p-js

**This repository has moved and is archived.**

`a7p-js` is now part of the [a7p](https://github.com/o-murphy/a7p) monorepo, which hosts the `.a7p` ballistic profile format implementation across multiple languages. The JavaScript/TypeScript package now lives at:

👉 **[github.com/o-murphy/a7p/tree/master/js](https://github.com/o-murphy/a7p/tree/master/js)**

This repository is archived (read-only) and no longer receives commits, issues, or pull requests. Please update your bookmarks, forks, and issue trackers to point to the new location. The `a7p-js` npm package will continue to be published from the new repository.

## Installation

```shell
yarn add a7p-js
# or
npm install a7p-js
```
